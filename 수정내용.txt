수정 완료 내용
- 로그인 후 첫 화면: /wholesale-ledger (도매 거래 한 줄 장부)
- 루트 주소 / 접속 시 도매 거래 한 줄 장부로 이동
- proxy.ts의 관리자 기본 이동 주소를 /dashboard에서 /wholesale-ledger로 변경
- 도매 한 줄 장부 페이지/API 권한 경로 추가
- dashboard 관련 화면 컴포넌트 제거
- .next 캐시 제거 (npm run dev 실행 시 자동 재생성)

사용 방법
1. 기존 erp 2 폴더를 백업해 두세요.
2. 이 압축을 풀어 나온 erp 2 폴더의 내용을 기존 프로젝트에 덮어쓰거나, 새 폴더로 통째로 사용하세요.
3. node_modules를 빼두었다면 다시 erp 2 폴더 안에 넣으세요.
4. VS Code에서 erp 2 폴더를 열고 npm run dev를 실행하세요.
5. http://localhost:3000 으로 접속하세요.
